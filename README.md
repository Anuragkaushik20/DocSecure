# DocSecure
Certificate verification desktop application that detects QR codes from uploaded certificates and validates their authenticity.

Features
Upload certificate image
Detect QR code
Extract verification link
Verify certificate authenticity
Tech Stack
Java (Swing) Python OpenCV / QR detection library

Architecture
Java Swing UI ↓ Python QR detection module ↓ Verification logic

How To Run
Clone repo
Install Python dependencies
Run Java application
Future Improvements
certificate search
database storage
verification history
